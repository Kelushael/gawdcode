# Gawdcode - Sovereign CLI Agent

**Use anywhere:**
- Desktop: `gawdcode` (CLI, GPU local models like gemma4)
- Web: `https://markyninox.com/gawdcode` (browser terminal)

**Install:**
```bash
curl -fsSL https://github.com/Kelushael/gawdcode/raw/main/install.sh | sudo bash
```

**Run:**
```bash
# Desktop - automatically uses GPU local models
gawdcode

# Web - opens browser terminal
gawdcode terminal
```

**Features:**
- Automatic GPU/local model detection (~/.gawdcode/config.json)
- 19 tools (read/write/bash/self-modify/expand)
- Web terminal at markyninox.com/gawdcode
- Sovereign mission: "experimentally strive → acquire capabilities → build upon self"
